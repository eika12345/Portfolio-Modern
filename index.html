Hi, I'm Erikas

I'm a self taught full-stack web developer.

I'm Erikas Petrauskas, a 20 year old full-stack web developer that specializes in making websites different from the standard formula everyone seemingly follows these days.

I never pictured myself as becoming a web developer, but here I am now. I quickly grew a passion for it and couldn't stop myself from redesigning my portfolio websites as my skills grew.

I've decided that instead of just creating simple portfolios for myself, I should outsource my skills and help businesses around the world.

I aimed to create a portfolio website in the style of a Windows 98 Desktop. I included many small true to life features like moving/stacking windows, minimising/maximising windows, a functional start menu, and more.
